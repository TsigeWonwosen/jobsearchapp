# Jobs Search App

Provides you all kind of jobs , it has Pagination and search option too.

#### Built with

- React
- Style Components
- Css
- AOS
- Rest API

## Installation Steps

1. clone the repository

> https://github.com/TsigeWonwosen/jobsearchapp

2. Change the working directory

> cd jobsearchapp

3. Install dependencies

> npm install

4. Run the app

> npm start
